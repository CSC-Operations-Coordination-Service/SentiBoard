# SentiBoard

## Installation

- Requirements : Python3.11 / Python3.10

```bash
python3.11 -m venv venv
source ./venv/bin/activate
```

```bash
pip install -r requirementx.txt
```

## Running dev mode

```bash
FLASK_DEBUG=True flask -e env.sample run
```

## Format

```bash
pip install black
black ./apps
```
